AssemblyCache Methods

The AssemblyCache type exposes the following members.

Methods
  NameDescription
InstallAssemblies
Install the provided assemblies to GAC
InstallAssembly
Install assembly into GAC
QueryAssemblyInfo
Query an assembly in the GAC. assemblyName has to be a fully specified name. E.g., for v1.0/v1.1 assemblies, it should be "name, Version=xx, Culture=xx, PublicKeyToken=xx". For v2.0+ assemblies, it should be "name, Version=xx, Culture=xx, PublicKeyToken=xx, ProcessorArchitecture=xx". If assemblyName is not fully specified, a random matching assembly may be found.
UninstallAssemblies
Uninstall the provided assembly names from the GAC. assemblyNames have to be fully specified names. E.g., for v1.0/v1.1 assemblies, it should be "name, Version=xx, Culture=xx, PublicKeyToken=xx". For v2.0+ assemblies, it should be "name, Version=xx, Culture=xx, PublicKeyToken=xx, ProcessorArchitecture=xx". If assemblyNames is not fully specified, a random matching assembly could be uninstalled.
UninstallAssembly
Uninstall an assembly from the GAC. assemblyName has to be fully specified name. E.g., for v1.0/v1.1 assemblies, it should be "name, Version=xx, Culture=xx, PublicKeyToken=xx". For v2.0+ assemblies, it should be "name, Version=xx, Culture=xx, PublicKeyToken=xx, ProcessorArchitecture=xx". If assemblyName is not fully specified, a random matching assembly could be uninstalled.
