AssemblyCacheUninstallAssembly Method

Uninstall an assembly from the GAC. assemblyName has to be fully specified name. E.g., for v1.0/v1.1 assemblies, it should be "name, Version=xx, Culture=xx, PublicKeyToken=xx". For v2.0+ assemblies, it should be "name, Version=xx, Culture=xx, PublicKeyToken=xx, ProcessorArchitecture=xx". If assemblyName is not fully specified, a random matching assembly could be uninstalled.
