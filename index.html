Welcome to EasyHook.

EasyHook starts where Microsoft Detours ends.

This project supports extending (hooking) unmanaged code (APIs) with pure managed ones, from within a fully managed environment on 32- or 64-bit Windows XP SP2, Windows Vista x64, Windows Server 2008 x64, Windows 7, and Windows 8.1.

Authors and Contributors

EasyHook was originally released by Christophe Husse in 2009 and since 2012 has been maintained by Justin Stenning (@spazzarama). The project moved from Codeplex to GitHub in August 2015.
